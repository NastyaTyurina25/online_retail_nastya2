# online_retail_nastya2

ABC-анализ ассортимента E-commerce (Online Retail II)

О проекте

Комплексный анализ товарного ассортимента британского онлайн-ритейлера с целью оптимизации складских запасов и повышения эффективности продаж. Проект демонстрирует навыки работы с базами данных (SQL), анализа больших объёмов данных и формулирования бизнес-рекомендаций на основе аналитических инсайтов.

Бизнес-проблема

Компания сталкивается с классической проблемой ритейла: 80% выручки генерируется 20% ассортимента, но при этом ресурсы (складские площади, логистика, маркетинговый бюджет) распределяются равномерно по всему товарному портфелю. Это приводит к:
- Замораживанию капитала в неликвидных товарах
- Росту затрат на хранение
- Снижению оборачиваемости запасов

Цель: Выявить товары-локомотивы (группа A), середняки (группа B) и аутсайдеры (группа C) для принятия решений.

Технологический стек

- PostgreSQL — загрузка и обработка данных, выполнение SQL-запросов
- SQL — оконные функции (Window Functions), CTE (Common Table Expressions), агрегация данных
- Excel — сводные таблицы (Pivot Tables), визуализация данных, расчёт процентов

 Методология

 1. Подготовка данных
- Загрузка датасета Online Retail II в PostgreSQL
- Очистка данных: фильтрация возвратов товаров (отрицательные значения `Quantity` и кредит-ноты с префиксом `C` в `InvoiceNo`)
- Расчёт чистой выручки по каждому товару: `TotalRevenue = Quantity × UnitPrice`

 2. ABC-анализ (SQL)
Реализован многошаговый SQL-запрос с использованием оконных функций:
- Шаг 1: Агрегация выручки и количества по каждому товару
- Шаг 2: Расчёт доли каждого товара в общей выручке (`RevenuePercent`)
- Шаг 3: Расчёт накопительного итога (`CumulativePercent`) с сортировкой по убыванию выручки
- Шаг 4: Присвоение ABC-категорий на основе правила Парето:
  - Группа A (Локомотивы): товары, формирующие 0–80% cumulative revenue
  - Группа B (Середняки): товары, формирующие 80–95% cumulative revenue
  - Группа C (Аутсайдеры): товары, формирующие 95–100% cumulative revenue

 3. Анализ результатов (Excel)
- Создание сводной таблицы для оценки масштаба: количество товаров и доля выручки по каждой группе
- Расчёт процентов от общего итога для наглядного сравнения вклада групп A, B и C

 Бизнес-рекомендации

 Для товаров группы A (Локомотивы):
1. Контроль остатков: Внедрить систему мониторинга stock levels в реальном времени. Out-of-stock по этим товарам недопустим — это прямые потери выручки.
2. Использовать высокие объёмы закупок для получения оптовых скидок и улучшения условий оплаты.
3. Маркетинговый фокус на товарах группы A в рекламных кампаниях и акциях для привлечения новых клиентов.

 Для товаров группы C (Аутсайдеры):
1. Оптимизация ассортимента: Рассмотреть вывод из постоянного ассортимента товаров с нулевой или минимальной оборачиваемостью.
2. Распродажа остатков: Провести ликвидацию запасов.
3. Снижение затрат: Освободившиеся складские площади использовать для расширения ассортимента товаров группы A или B.

 Для товаров группы B (Середняки):
1. Стимулирование продаж: Провести анализ, какие товары из группы B имеют потенциал перехода в группу A (например, через улучшение карточек товаров, маркетинговые акции).
2. Кросс-продажи: Предлагать товары группы B в комплекте с товарами группы A для увеличения среднего чека.
